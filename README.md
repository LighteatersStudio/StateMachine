# Async State Machine

An asynchronous state machine framework for Unity built on Task and async/await principles.

## Requirements

- Unity 2020.3 or higher
- Zenject/Extenject

## Installation

### Step 1: Install Extenject

Open Unity Package Manager (`Window â†’ Package Manager`), click the **+** button, select **Add package from git URL**, and enter:
```
https://github.com/svermeulen/Extenject.git?path=/UnityProject/Assets/Plugins/Zenject
```

### Step 2: Install State Machine

Click **+** again and enter:
```
https://github.com/LighteatersStudio/StateMachine.git
```
